# Developing

Fetch submodules

    git submodule init
    git submodule update

Install deps for dev modules

    cd dev_node_modules/piles/
    npm install --dev

Install deps on this repo root:

    npm install --dev

Run server

    npm start

Develop server

    bin/develop

To test production mode

    NODE_ENV=production npm start

Deploy

    bin/deploy linode
